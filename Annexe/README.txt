ARCHITECTURE DES ORDINATEURS - Devoir Chapitres 2 à 5

Ce dossier contient :
- Le PDF regroupant toutes les réponses aux exercices des chapitres 2 à 5.
- Une annexe avec les fichiers Python correspondant aux exercices réalisés.

CONTENU DE L’ANNEXE :

Chapitre 2 :
- Chap_2_ex_2.2.py
  Contient les exercices 2.2 et 2.3.
  * Exercice 2.2 : Décrire la suite d’opérations pour additionner deux nombres en chiffres romains et écrire le programme en Python.
  * Exercice 2.3 : Programme Python qui affiche un nombre saisi par l’utilisateur en chiffres romains.
  > Chaque exercice est identifié par un commentaire dans le fichier.

Chapitre 3 :
- Chap_3_ex_3.14.py
  Programme Python pour comparer deux nombres représentés en complément à 2 et déterminer lequel est le plus grand en utilisant uniquement des comparaisons de bits.
  > La saisie des nombres peut se faire par l’utilisateur

Chapitre 4 :
- Chap_4_ex_4.13.py
  Programme Python pour multiplier deux nombres à virgule flottante saisis par l’utilisateur.

- Chap_4_ex_4.17.py
  Programme Python qui calcule la fonction exponentielle selon la formule de Taylor.

INSTRUCTIONS D’UTILISATION :

1. PDF principal :
   - Ouvrir le fichier PDF pour consulter toutes les réponses et exercices résolus.

2. Fichiers .py :
   - Aucun module externe n’est nécessaire.
   - Tout est réalisé avec python comme langage
